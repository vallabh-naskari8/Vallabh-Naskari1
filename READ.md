# Vallabh Naskari - Tandemloop Screening Test Solutions


## Problem 1: Calculator
- **File:** Problem-1.js  
- **Description:**  
  Implements a simple calculator using a class.  
  Supports operations: Addition (`add`), Subtraction (`sub`), Multiplication (`mul`), Division (`div`), Modulo (`mod`).  
- **Input:** `a` (number), `b` (number), `operation` (string)  
- **Output:** Result of the operation. Handles divide/modulo by zero.  

---

## Problem 2: Odd Number Series
- **File:** Problem-2.js  
- **Description:**  
  Generates the first `a` odd numbers starting from 1.  
- **Input:** Single integer `a`  
- **Output:** Array of first `a` odd numbers  

---

## Problem 3: Conditional Odd Number Series
- **File:** Problem-3.js  
- **Description:**  
  Generates a series of odd numbers based on the input:  
  - If `a` is odd → generates `a` odd numbers  
  - If `a` is even → generates `a-1` odd numbers  
- **Input:** Single integer `a`  
- **Output:** Array of odd numbers  

---

## Problem 4: Count Multiples
- **File:** Problem-4.js  
- **Description:**  
  Counts how many numbers in an array are divisible by 1 to 9.  
- **Input:** Array of integers  
- **Output:** Object/dictionary showing counts for divisors 1–9
